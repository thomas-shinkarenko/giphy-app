Go into backend folder
```cd giphy-backend```

Run docker compose up to exec and startup the container
```docker-compose up -d```

Install dependencies
```yarn install```

Execute the server
```yarn dev```


Go into backend folder
```cd giphy-frontend```

Install dependencies with yarn or one of your preference
```yarn install```

Execute the server
```yarn dev```

Go to
http://localhost:5173/
Enjoy :D
